## Running the code

**Prerequisites:** Node.js 22+ (for native TypeScript support via `--experimental-strip-types`).

**Run directly:**
```bash
npm start                  # Runs main.ts directly
```

**Development mode (watch for changes):**
```bash
npm run dev                # Runs with --watch flag
```

**Type check:**
```bash
npm run check              # Runs tsc for type checking
```

**File structure:**
- `lib.ts` - Library functions
- `main.ts` - Main entry point
