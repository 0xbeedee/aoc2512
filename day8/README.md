## Running the code

**Prerequisites:** You need Elixir installed. Get it from [elixir-lang.org](https://elixir-lang.org/install.html).

**Build and run:**
```bash
mix compile                # Compiles the project
mix run -e "Day8.Main.main"  # Runs the main function
```

**Run tests:**
```bash
mix test                   # Runs all tests
```

**Clean up:**
```bash
mix clean                  # Removes build artifacts
```

The project uses Elixir with Mix build tool.

**File structure:**
- `lib/day8.ex` - Core module with library functions
- `lib/main.ex` - Main entry point
