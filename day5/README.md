## Running the code

**Prerequisites:** You need Java Development Kit (JDK) installed (Java 8 or later).

**Build and run:**
```bash
javac Main.java Lib.java Range.java    # Compiles the source files into .class files
java Main                               # Runs the program
```

**Clean up:**
```bash
rm *.class                              # Removes compiled .class files
```

The project uses Java 25.0.1.
